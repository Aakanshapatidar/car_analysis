# 🚗 Used Car Sales Analysis Dashboard

## 📌 Project Overview

This project presents an end-to-end **Data Analytics** workflow on a **1,000,000-record Used Car dataset** using **Python, PostgreSQL, SQL, and Power BI**.

The project focuses on cleaning raw data, performing SQL-based business analysis, and building interactive Power BI dashboards to generate actionable business insights.

---

## 🎯 Objectives

- Clean and preprocess raw data using Python
- Store the cleaned dataset in PostgreSQL
- Perform SQL-based business analysis
- Build interactive Power BI dashboards
- Generate meaningful business insights for decision-making

# 🧹 Data Cleaning (Python)

The dataset was cleaned using Python (Pandas).

### Cleaning Steps

- Removed duplicate records
- Handled missing values
- Filled numerical values using **Median**
- Filled categorical values using **Mode**
- Standardized brand names
- Standardized fuel type names
- Corrected inconsistent values
- Exported cleaned dataset

# 📈 Key Insights

- 🚗 Total Cars: **1,000,000**
- 🏷️ Total Brands: **13**
- 💰 Average Price: **₹1,012,877**
- ⭐ Mercedes has the highest average selling price.
- ⛽ Petrol is the most common fuel type.

- 📉 Older vehicles generally have lower resale prices.
